# 大平自動車商会 - 顧客管理システム

## 概要
大平自動車商会の顧客データを管理し、車検期限案内を自動化するWebアプリケーションです。

## 機能
- **顧客データ管理**: Excelファイルから顧客情報を読み込み、一覧表示
- **車検期限案内**: 今月から翌々月までの車検期限が近い顧客を自動抽出
- **案内文書作成**: はがきサイズの案内文書を一括生成・印刷
- **検索・ソート**: 顧客データの検索と並び替え機能
- **レスポンシブ対応**: スマートフォンでも利用可能

## 技術スタック
- **フロントエンド**: HTML5, CSS3, JavaScript (Vanilla)
- **バックエンド**: Node.js, Express.js
- **データ処理**: xlsx (Excelファイル読み込み)
- **言語**: TypeScript

## セットアップ

### 前提条件
- Node.js (v14以上)
- npm

### インストール
```bash
cd webapp
npm install
```

### 開発サーバー起動
```bash
npm run dev
```

### 本番ビルド
```bash
npm run build
npm start
```

## 使用方法

1. アプリケーションを起動後、ブラウザで `http://localhost:3000` にアクセス
2. **全顧客一覧**: 全ての顧客データを表示・検索
3. **車検案内**: 車検期限が近い顧客のみを表示
4. **案内文書作成**: はがきサイズの案内文書を生成・印刷

## データ構造
顧客データは `☆顧客フォルダ` 内の各サブフォルダの `顧客ファイル.xlsx` から読み込まれます。

## ライセンス
Private use only
