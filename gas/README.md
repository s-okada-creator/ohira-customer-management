# 顧客データ管理システム - Google Apps Script

顧客フォルダ統合Excelファイル用のGoogle Apps Script（GAS）コードです。

## 📋 機能一覧

### 1. メニューバー機能
- **新規顧客を追加**: フォームから新規顧客情報を入力して、自動的にシートを作成
- **顧客を検索**: キーワードで顧客を検索し、該当するシートに移動
- **統計情報を表示**: 顧客数やシート数の統計情報を表示

### 2. 新規顧客追加機能
- 自動的な顧客番号生成（4桁ゼロパディング）
- 顧客名簿シートの自動作成
- 顧客データベースシートの自動作成
- 目次シートの自動更新
- 処理ログの記録
- 作成されたシートへの自動移動

### 3. 検索機能
- 顧客名、車種名、顧客番号での検索
- 検索結果から直接シートに移動可能
- リアルタイム検索結果表示

### 4. 統計機能
- 総顧客数の表示
- 総シート数の表示
- 最終更新日時の表示

## 🚀 セットアップ手順

### 1. Google スプレッドシートにコードを追加

1. 統合されたExcelファイルをGoogle スプレッドシートにアップロード
2. `拡張機能` → `Apps Script` を選択
3. 以下のファイルを追加：
   - `Code.gs` - メインスクリプト
   - `AddCustomerDialog.html` - 新規顧客追加ダイアログ
   - `SearchCustomerDialog.html` - 顧客検索ダイアログ

### 2. 権限の設定

1. 初回実行時に権限の許可を求められるので、許可する
2. 以下の権限が必要です：
   - スプレッドシートの閲覧・編集
   - ダイアログとサイドバーの表示

### 3. スプレッドシートを開く

スプレッドシートを開くと、自動的に「顧客管理」メニューが表示されます。

## 📝 使い方

### 新規顧客を追加

1. メニューバーから `顧客管理` → `新規顧客を追加` を選択
2. ダイアログで顧客情報を入力：
   - **基本情報**: 顧客名、ふりがな、住所、連絡先など
   - **車両情報**: 車種名、ナンバー、年式、色など
   - **その他情報**: 担当者、紹介者、備考など
3. `顧客を追加` ボタンをクリック
4. 自動的に以下が実行されます：
   - 新しい顧客番号が生成される
   - 顧客名簿シートと顧客データベースシートが作成される
   - 目次シートに情報が追加される
   - 処理ログが記録される

### 顧客を検索

1. メニューバーから `顧客管理` → `顧客を検索` を選択
2. 検索キーワードを入力（顧客名、車種名、顧客番号）
3. `検索` ボタンをクリック
4. 検索結果から顧客をクリックすると、該当するシートに移動

### 統計情報を表示

1. メニューバーから `顧客管理` → `統計情報を表示` を選択
2. ポップアップで統計情報が表示される

## 🔧 カスタマイズ

### シート名の形式を変更

`createCustomerNameSheet` 関数と `createCustomerDatabaseSheet` 関数の以下の部分を編集：

```javascript
const sheetName = `${customerNumber}_${customerData.customerName}_${customerData.carName}_顧客名簿`;
```

### フォーム項目の追加

`AddCustomerDialog.html` に新しいフォーム項目を追加し、`createCustomerNameSheet` 関数と `createCustomerDatabaseSheet` 関数で対応するデータ処理を追加します。

## ⚠️ 注意事項

- 顧客番号は目次シートの情報から自動生成されるため、目次シートを削除しないでください
- 大量のシートを作成する場合、処理に時間がかかることがあります
- Excelファイルから変換する際、一部の書式が失われる可能性があります

## 📊 データ構造

### 顧客名簿シート
- 顧客の基本情報を見やすい形式で表示
- 印刷用のレイアウト

### 顧客データベースシート
- 全ての顧客情報を1行で管理
- データ分析や検索に適した形式
- タイムスタンプを自動記録

### 目次シート
- 全顧客の一覧
- 顧客番号、顧客名、車種名、シート名、ファイルパスを記録

### 処理ログシート
- 全ての操作を記録
- 処理日時、処理内容、顧客番号、詳細を保存

## 🔐 セキュリティ

- Google Apps Scriptはサーバーサイドで実行されるため、機密情報が安全に保管されます
- スプレッドシートの共有設定により、アクセス権限を管理できます
- 処理ログにより、全ての操作が追跡可能です

## 📞 サポート

問題が発生した場合は、処理ログシートを確認するか、Apps Script エディタのログを確認してください。
